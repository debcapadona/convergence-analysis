# Convergence Analysis

Societal instability dashboard tracking 14 threads of structural stress and counterweight data.

Built on Turchin structural-demographic theory. Each thread shows pressure and slack simultaneously.

## Stack
- Frontend: React + Recharts, hosted on GitHub Pages
- Fetchers: Python 3.11+, scheduled via GitHub Actions
- Data: JSON files per thread, schema-validated on every write
- Future: Postgres swap-in without touching fetchers or components
